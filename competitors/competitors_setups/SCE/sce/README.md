# Sparsest-Cut-Network-Embedding
Code for the KDD 2020 paper "[SCE: Scalable Network Embedding from Sparsest Cut](https://arxiv.org/abs/2006.16499)".


## Requirements
* python == 3.7
* pytorch == 1.1.0  
* numpy == 1.17.2
* scikit-learn== 0.19.1
